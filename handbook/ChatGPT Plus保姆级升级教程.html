    # ChatGPT Plus保姆级升级教程

    # 如何判断自己是否需要升级Plus

    相比3.5版本，Plus的回复更快、更有质量、对问题的理解更深刻以及回答更有逻辑。

    如果你是需要高频率使用或者说需要更高质量回复的选手，那么最好升级。

    如果你是职业可提升性很高，如程序猿、自媒体、产品经理等等，对你的工作大有帮助的，也建议升级。

    类似学生或很少能用到的人，使用免费的3.5也够用了。当然，花个20美金开一个月来试试也是很值的。

    # Plus的优势是什么

    • Plus回答质量毫无疑问的比3.5更强，也是目前世界上最好用的GPT

    • Plus可以联网搜集数据再回答，而3.5的数据停留至2021年

    • Plus支持图片输入、分析图片

    • Plus拥有一个插件市场，这相当于AppStore对于苹果手机的作用，可以发挥更多更大的作用

      另外，按以往风格，内测名额都是开在Plus里，如果对sora感兴趣的那就得提前升级好了

    # 三种升级方式

    大概有三种方式：

    1. 绑定国外信用卡或虚拟卡上官网升级
    2. 使用App Store礼品卡在手机端升级（据24.2.19有人反馈礼品卡维护）
    3. 使用国外平台WildCard一键升级（推荐，实测几分钟解决）

    （PS：如果不会操作，也可以提供代充、GPT 账号注册等服务，有需要加微信：572874317，备注：chatgpt

    # WildCard

    进入平台后可以一键升级Plus，具体详情可以在我另一篇文章中看：

    《手把手教学，如何开通 ChatGPT Plus: GPT4升级指南》 (opens new window)

    # 绑定国外信用卡（或者虚拟信用卡）

    需知：

    1. 国内所有VISA和MasterCard的信用卡，即使有外币支付功能，也绑定不上；
    2. 香港信用卡、香港虚拟信用卡同样无法绑定。

    首先先进入GPT首页，点击Upgrade to Plus

    然后填上你的国外信用卡信息，就可以点击Subscribe提交了

    # 苹果礼品卡（被维护了，暂不可行）

    大致流程是：通过礼品卡把钱充到Apple ID里面，然后再用Apple ID购买ChatGPT Plus套餐

    注意：

    在使用支付宝购买礼品卡时，可能会遇到黑卡导致账号被封的情况。如果遇到这种情况，建议从苹果官网购买官方礼品卡进行充值，以确保账号的安全。

    # 支付宝购买AppStore礼品卡

    1. 切换地址为旧金山
    2. 搜索或者在新显示的小程序里点击出镜
    3. 点击折扣礼卡
    4. 点击更多大牌折扣礼卡，搜索AppStore

    # 充值进自己的苹果账户

    1. 进入App Store
    2. 点击自己头像进入个人信息页
    3. 选择【兑换充值卡或者代码】并输入
    4. 兑换，以及查看金额是否到账

    如果有问题及时带着AppleID咨询苹果客服，效率很高。很多同学都是十分钟内解决的问题

    # 学习GPT的途径

    个人推荐的是知识星球里的破局俱乐部

    在里面我也是默默潜水的普通人。但是它提供的交付绝对是国内顶尖的，而且一年只需要 3、400 左右，对于其他动辄 500+ 或 上千的 AI 课程绝对是良心价。

    仅我见到的教程就有抖音爆火过的 机车赛博朋克变装、图片高光艺术字 和 AI绘画、GPT提示词大航海。

    购买三天内无条件退款，其他就见仁见智了，不再多言。
